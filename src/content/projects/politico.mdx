# Politico

## **Overview**

An interactive political engagement platform developed for Politico that combines modern web technologies with gamification elements to create compelling user experiences during election seasons.

## **Key Features**

- **Interactive Contest Platform** - Built an engaging contest system with special touches of interactivity and gamification
- **Custom Scoring Algorithm** - Developed a sophisticated algorithm to calculate user scores based on their submissions
- **Election Season Launch** - Platform was strategically unveiled during the midterm election season
- **Social Media Integration** - Promoted extensively across Politico's website and social media channels

## **Technical Implementation**

- **HTML** - Semantic markup for accessibility and SEO
- **CSS** - Modern styling with PostCSS for enhanced browser compatibility
- **Sass** - Modular stylesheets for maintainable code architecture
- **JavaScript** - Interactive features and dynamic user engagement
- **Custom Algorithm Development** - Backend logic for real-time score calculations

The project successfully merged political content with interactive technology, creating an engaging platform that encouraged user participation during a critical election period.
